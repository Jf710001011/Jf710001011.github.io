内存分区模型

  • 代码区：二进制代码
    • 运行前，共享，只读
  • 全局区：全局变量、静态变量、全局常量
    • 运行前，操作系统释放
  • 栈区：参数值、局部变量，编译器自动分配释放
    • 不要返回局部变量的地址
  • 堆区：由程序员分配和释放，若程序员不释放，则由操作系统回收
    • new开辟内存

引用

  • 引用必须要初始化，一旦初始化，便不可更改（是谁的别名），可以赋值
    • 不要返回局部变量的引用
    • 函数的调用可以成为左值
    • 引用的本质：指针常量
    • 自动解引用
    • 常量引用:用在函数参数中，防止修改值
      • const int& ref = 10的本质：创建int temp;

函数提高

  • 默认参数 > 从这个默认值往后都必须有默认值 > 或者声明有默认参数，或者实现有默认参数，不可同时有

  • 函数占位参数:可以有默认参数，调用时参数必须填补 

    void fuction(int a, int){
    cout << "this is function " << endl;
}

  • 函数重载: 函数名可以相同 > 同一个作用域下 > 函数名称相同 > 函数类型不同 或 参数个数不同 或 参数顺序不同 > 返回值不能作为重载的条件 > int a = 10;func(a)(引用);func(10)(const 引用)，有无const 也能发生重载 > 函数重载遇上默认参数

面向对象

  • 访问权限 > public 类内/外都可以访问 > protected 类内可以，类外不可以 > private 类内可以，类外不可以 > struct默认公有访问权限，class默认私有访问权限 > 成员属性设置为私有：1. 可以自己控制读写权限。2.对于写可以控制数据的有效性。在方法上操作

  • .h 与.cpp文件的分工

  • 构造函数与析构函数 > 构造函数可以重载 > 拷贝构造 > 析构函数没有参数 > 匿名对象Person(10),当前执行结束后，会立即收回匿名对象，不要用拷贝构造函数初始化匿名对象（重定义） > 拷贝构造函数：1.值传递给函数传参。2.以值方式返回局部对象 > 默认构造函数、默认析构函数、默认拷贝构造函数 > - 构造函数调用规则：1. 如果用户定义有参构造函数， 则c++不提供默认构造函数，但提供默认拷贝构造函数。2. 如果用户定义拷贝构造函数，C++不会提供默认构造函数。

    析构函数：将堆区开辟数据释放 浅拷贝：堆区的内存重复释放。深拷贝解决

  • 初始化列表 > Person(int a,int b,int c):A(a),B(b),C(c){} > 先构造类对象，再构造自身，析构顺序与之相反

  • 静态成员 > 所有对象共享 > 静态成员函数只能访问静态成员：1. 可以类名访问

  • 对象特性 > 空对象占用一个字节 > 只有非静态成员属于类的对象 > 返回对象 与 返回对象的引用 > 空指针访问成员函数

  • 常函数void ShowPerson()const{}不可以修改成员属性，除非变量前加mutable

  • 常对象只能调用常函数

  • thi指针的本质是指针常量，指向不可修改

  • 友元：让一个函数或类访问另一个类中私有成员 > - 全局函数做友元:在Building类中friend void GoodGuy(Building& building) > - 类做友元： 在Building类中friend class GoodGuy; > - 成员函数做友元:在Building类中friend void Goodguy::visit()

  • 运算符重载 > - 成员函数重载 或 全局函数重载 > Person operator+() > 只能利用全局函数重载<<：ostream& operator <<(ostream& out,Person& p) > 区分重置前置++与后置++：使用占位参数。后置递增返回值而不是引用：返回的临时变量不用引用返回 > - 函数调用运算符重载：仿函数

三大特性

  • 封装

  • 继承:class 子类：继承方式 父类,减少重复代码 > class java: public BasePage{} > 父类中所有非静态成员属性都会被继承 > 开发人员命令提示符：查看对象模型：1.跳转文件路径 2. cl /d1 reportSingleClassLayout类名 文件名 > 构造函数先父类后子类，析构顺序正好相反 > 继承同名：子类的直接访问，父类的加作用域。 son s;s.m_a,s.base::m_a,同名函数同理 > 如果子类中出现和父类同名的成员函数，子类的同名成员函数会隐藏掉父类中所有同名成员函数 > source\0_6.png：animal->(sheep,tuo)->yangtuo,资源浪费 > - 虚基类 > - vbptr vbtable

  • 多态 > 静态多态：函数重载，运算符重载，编译阶段确定函数地址 > 动态多态：派生类和虚函数，运行阶段确定函数地址 > 1. 继承关系 2. 子类重写父类的虚函数 > 使用：父类的指针 或 引用 指向子类对象（ 函数参数） > vfptr: 当子类重写虚函数后，子类中的虚函数表（记录函数地址）中的内容会更新，会覆盖原先继承的来自父类的表

    计算器类：对扩展进行开放，对修改进行关闭

  • 纯虚函数 > 基类中纯虚函数的实现毫无意义，抽象类：含有纯虚函数的类，无法实例化对象。纯虚函数： virtual 返回值 函数名（） = 0 > 虚析构函数：解决父类指针释放子类对象时不干净的问题 > 虚析构和纯虚析构：都需要在父类中有实现。含纯虚析构函数的类为抽象类，无法实例化

  • 写文件 > - 创建流对象ofstream ofs; > - 打开文件ofs.open("文件路径"，打开方式) > - ofs << "写入的数据" > - 关闭文件

#include<iostream>
#include<fstream>
using namespace std;

void test01() {
	ofstream ofs;
	ofs.open("test.txt", ios::out);

	ofs << "姓名：张三" << endl;
	ofs << "性别：男" << endl;
	ofs << "年龄：18" << endl;

	ofs.close();
}

int main() {
	test01();

	system("pause");

	return 0;
}
